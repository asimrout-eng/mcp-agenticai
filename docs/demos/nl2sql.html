Get the Natural Language to SQL demo running locally

1

Prerequisites

  • Docker Desktop installed and running
  • Python 3.8+ with pip
  • Anthropic API key (Claude access)
  • Firebolt account with service account credentials
2

Clone and Install

git clone https://github.com/asimrout-eng/mcp-agenticai.git cd mcp-nl-devrel pip install -r requirements.txt
3

Configure Credentials

# Set your Anthropic API key export ANTHROPIC_API_KEY="your_api_key_here" # Configure Firebolt credentials (optional - can set in UI) export FIREBOLT_MCP_CLIENT_ID="your_service_account_id" export FIREBOLT_MCP_CLIENT_SECRET="your_service_account_secret"
4

Launch Demo

# Run the Natural Language to SQL demo ./run.sh your_anthropic_api_key # Access at: http://localhost:8504

Demo Tips

  • Connection: Use the UI to enter Firebolt credentials or set environment variables
  • Questions: Start with simple queries and progress to complex multi-table joins
  • SQL Editing: Review and modify generated SQL before execution
  • Visualizations: Experiment with different chart types for your data
  • Performance: Check Firebolt engine times to see sub-second execution
